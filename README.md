# Project Title

## Movie Rating System Using MERN Stack

---

# Project Description

Movie Rating System is a full-stack web application that allows users to browse movies, view details, rate movies, and submit reviews. The platform provides an interactive user experience for discovering and reviewing movies.

The application demonstrates MERN stack development, authentication, database management, API development, and responsive UI design.

---

# Project Scope

The project aims to provide a platform where users can:

* Browse movie collections
* View movie details
* Rate movies
* Submit reviews
* Search movies
* Manage user profiles

---

# Features

* User Registration & Login
* JWT Authentication
* Browse Movies
* Search Movies
* Movie Details Page
* Rating System
* User Reviews
* Responsive Design
* MongoDB Database Integration
* REST API Architecture
